== Description == 

Automatically, cross-posting to associated site/blog to enabling the post-via-email option with PIN code e.g. multiply.com, livejournal.com, blogspot.com or even other wordpress-base site and much more.

Please see the CrossPress detail [CrossPress Project](http://www.atthakorn.com/project/crosspress/ "CrossPress Project Page")
